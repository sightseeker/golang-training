Golang Traning
===
## System Requirements
* Go 1.5+
* Glide (Golang Package Manager)

## Build
```
go get github.com/sightseeker/golang-training   # Get this package
glide install                                   # Install packages and dependencies
go build                                        # Build this package
```

## Running API

```
./golang-training
```

## Access API

http://localhost:1323
